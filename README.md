## ember-cli-esnext

This addon inserts adds `esnext` processing into the javascript preprocessor for Ember CLI.

### Installation / Usage

From within your Ember CLI application (must be > 0.0.34), run the following:

```bash
npm install --save-dev ember-cli-esnext
```
